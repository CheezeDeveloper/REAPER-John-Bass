### Features
*   **Massive Bass Boost:** Heavy distortion and gain.
*   **Destruction Amount:** Adjustable control from subtle to extreme.
*   **Presets:** Includes Mild Fry, Deep Fried Meme, Broken Android, and THE NUCLEAR OPTION.
*   **Low Quality Mode:** Bitcrushing effect.
*   **The Ear Piercer:** Optional high-frequency beep (3000Hz).

### Installation
1.  Open REAPER.
2.  Go to **Options** ‚Üí **Show REAPER resource path**.
3.  Open the **Effects** folder.
4.  Drop the `John Bass` folder inside.
5.  Restart REAPER or scan for new plugins.
